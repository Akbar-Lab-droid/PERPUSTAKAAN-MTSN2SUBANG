# 📚 Perpustakaan Digital MTSN 2 Subang
Selamat datang di proyek Perpustakaan Digital MTSN 2 Subang! ✨ Proyek inovatif ini dirancang untuk mendigitalkan dan mengoptimalkan pengelolaan koleksi buku serta seluruh aktivitas peminjaman di perpustakaan kami. Tujuannya sederhana: meningkatkan efisiensi, mempermudah akses informasi, dan memberikan pengalaman yang lebih baik bagi siswa dan staf. 🚀
🌟 Fitur Unggulan
Kami menghadirkan serangkaian fitur canggih untuk menyempurnakan operasional perpustakaan:
 * 📖 Manajemen Koleksi: Tambah, edit, dan hapus data buku dengan detail lengkap (judul, penulis, ISBN, kategori, status stok).
 * 🧑‍🏫 Manajemen Anggota: Kelola data anggota perpustakaan (siswa dan guru) secara terpusat.
 * 🤝 Transaksi Peminjaman & Pengembalian: Sistem otomatis untuk mencatat setiap transaksi peminjaman dan pengembalian buku.
 * 🔍 Pencarian Cepat & Filter: Temukan buku yang Anda butuhkan dalam hitungan detik dengan fitur pencarian berdasarkan judul, penulis, atau ISBN, dilengkapi opsi filter.
 * 📊 Laporan Dinamis: Hasilkan laporan visual untuk data peminjaman, buku terpopuler, dan status stok terkini.
